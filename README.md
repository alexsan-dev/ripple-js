# ripple-js
## Material design ripple effect with JavaScript

Download 
```
git clone https://github.com/alexsan-dev/ripple-js.git
```

Install npm 🔧
```
npm i ripple-js
```

CDN script tag
```javascript
<script href="https://raw.githubusercontent.com/alexsan-dev/ripple-js/master/index.js"></script>
```


Using ⚒️
```javascript
import 'ripple-js';
// on HTML or JSX
<div class="waves"></div>
```
